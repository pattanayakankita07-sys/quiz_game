# Quiz Game (C Project)

## Description
This is a console-based Quiz Game developed in the C programming language. It allows users to answer multiple-choice questions, tracks the score, and provides immediate feedback on answers. The game helps users practice and test their knowledge in a fun and interactive manner.

## Features
- Multiple-choice questions with 4 options each
- Score tracking with final score display
- Immediate feedback on correct or wrong answers
- Simple menu-driven interface for ease of use

## How to Run
1. Clone the repository:git clone https://github.com/pattanayakankita07-sys/quiz_game.git
2. Navigate to the project directory.
3. Compile the source code:gcc quiz_game.c -o quiz_game
4.  4. Run the executable:./quiz_game
